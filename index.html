Ceci est une aide afin de faciliter votre utilisation du site.

Description des fonctionnalités

Le bouton permet d'afficher un message de bienvenue.

Charger un fichier texte.

Pour commencer, il faut donner un fichier texte via le bouton

Le texte chargé sera affiché à gauche.

Segmenter le texte chargé.

Ensuite, on clique sur le bouton pour afficher le texte segmenté.

Le texte segmenté sera affiché à droite.

Compter les mots.

Pour avoir le comptage des mots, triés du plus fréquent au moins fréquent, on clique sur le bouton

Le tableau sera affiché à droite.

Afficher les lignes qui contiennent un motif.

Pour afficher les lignes qui contiennent un motif, on clique sur le bouton

Il faut avoir chargé un fichier en mémoire et renseigné le champs .

Grep est une commande historique de Unix ayant cette fonctionnalité. Nous employons ce nom en hommage.

Les lignes seront affichées à droite.

Pour chercher un terme dans le texte et de l'afficher en contexte, on clique sur le bouton

Le tableau sera affiché à droite.

Autres fonctionnalités utiles

Pour lire le texte à l'envers, on clique sur le bouton

Le tableau sera affiché à droite.

Pour savoir, qu'elle est le mot le plus long dans tout le texte, on clique sur le bouton

Le tableau sera affiché à droite.

Le bouton va afficher un graphique en barres de la fréquence des 4 premiers mots de l'ensemble du texte
